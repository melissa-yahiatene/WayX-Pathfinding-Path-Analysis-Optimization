===========================================
                WayX
===========================================

Auteure :
    YAHIATENE Melissa

Description :
    WayX est une application Java permettant d’exécuter des algorithmes
    de plus court chemin (Dijkstra et A*) sur :

        • des cartes .txt (mode Graphe)
        • des images (mode Image)

    L’application propose une interface graphique permettant :
        • de charger une carte ou une image
        • de visualiser le chemin optimal entre un point de départ et un point d’arrivée


===========================================
2) EXÉCUTION DU PROGRAMME
===========================================

1. Télécharger le fichier WayX.jar fourni dans le rendu.

2. Placer WayX.jar dans un dossier de votre choix.

3. Ouvrir un terminal (cmd, PowerShell ou terminal Linux)
   et se placer dans le dossier contenant WayX.jar :

       cd chemin/vers/MonDossierWayX

4. Lancer l’application avec la commande :

       java -jar WayX.jar

Le programme démarre alors sur le menu principal permettant de choisir :
    • Mode Graphe
    • Mode Image
